Setting Up The Admin panel

After login you will see the admin dashboard.click on the settings option in admin panel sidebar then you will reach to the Settings page. on settings page you have to changes credentials in some section with your credentials(SMS/OTP Settings, FCM Keys, Payment Mode,Map Settings) or have to make changes according to your need in some sections(App Notice, Driver Incentive(per Order), Global Settings).the settings page will look like below-
=====================================================================
For Map Setting Follow the below procedure(you can use google map or mapbox)-

For creating Google Map-
create a google map api key from Google console account and place it in admin panel Settings->Map Settings->Google Map . 

When your Google Maps Api key works successfully, the problem with Add delivery boy, Add store and Add area section will be solved. You have to create google map api key by your own via below link- https://console.cloud.google.com/google/maps-apis/overview

Note- Add card in Billing section is mandatory when you create google map api key. Check your google map api key with this link.

https://maps.googleapis.com/maps/api/geocode/json?latlng=-27.0000,133.0000&key=Place your google map api key here

see the attached screenshot i you will get response like that it means your map api key is working.otherwise your map api key is not working.

 
 
for creating mapbox api-
you can use mapbox as an alternative to Google map
create an account via https://account.mapbox.com/
after successful account creation and login you will get below page-

you mapbox token has been generated and you can find it under Access Tokens -> Default public token
=====================================================================
FCM Server Key Creation-
create a server key from firebase and put it on Admin panel Settings->FCM Server Key
User App FCM Server Key = for customer app,
Store App FCM Server Key = For store app
Driver App FCM Server Key = For Driver app
=====================================================================
Choose a Message gateway
choose a message gateway via Settings->SMS/OTP settings.
you can choose between two message gateways msg91 or twilio .

NOTE- if you choose msg91 then you have to buy transaction sms package of msg91.


If you don’t want to use any message gateways for the project then you simply choose OTP/Message Off from dropdown in settings->sms/OTP settings.

if you choose firebase otp on……. than otp will be sent via firebase.
=====================================================================

Setup SMTP details
sir you have to setup your SMTP from source/.env 
MAIL_DRIVER=smtp
MAIL_HOST=smtp.gmail.com
MAIL_PORT=465
MAIL_USERNAME=smtp mail
MAIL_PASSWORD= smtp password
MAIL_ENCRYPTION=ssl


You can create gmail smtp credentials by following the steps from below link-

All Set Up

If you follow the all steps successfully then your backend is ready for live now-

Your apps base url and Img Url will be follows-

User App-  domain_name/folder_name/api/

Store App-  domain_name/folder_name/api/store/

Driver App- domain_name/folder_name/api/driver/

IMG URL for all Apps-  domain_name/folder_name/
